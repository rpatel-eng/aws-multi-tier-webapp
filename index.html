AWS Multi-Tier Web Application

Project Overview

This project demonstrates a professional AWS multi-tier web application with:

  • VPC with public/private subnets and NAT gateway
  • EC2 web servers running Apache & PHP
  • RDS MySQL database in private subnet
  • Auto Scaling & Application Load Balancer (ALB)
  • CloudFront CDN for content delivery
  • Secure PHP application with input validation & exception handling
